GNU Radio Tutorials

Beginner GNU Radio tutorials

  • Introducing GNU Radio has information on GNU Radio and the installation process.
  • Flowgraph Fundamentals gets you started understanding flowgraphs
  • Creating and Modifying Python Blocks. This section teaches you how to make your own blocks for GNU Radio flowgaphs. (make sure you know where the python blocks are being saved so you can open the files and edit it with your favourite text editor).
  • The DSP blocks section introduces some blocks useful for digital signal processing.

    • Intermediate/Advanced GNU Radio tutorials

  • Read through PMTs, message passing, virtual sources and virtual sinks. These are important when making complex flowgraphs.
  • Mod and Demod section teaches you about different modulation and demodulation techniques such as QPSK, BPSK. It also teaches about multiplexing like OFDM (orthogonal frequency division multiplexing).

    • Using an SDR with GNU Radio

    An SDR can be a receiver or a transceiver. Using an SDR with GNU Radio you can receive signals and maybe even transmit signals.
    Some SDRs are quite expensive, for example, Ettus Research's USRP. You can get a cheaper one like the RTL SDR. The RTL SDR will cost
    around $50. Note that the RTL SDR is just a receiver, it cannot transmit signals.

  • Start with the hardware tutorial to listen for signals. Then you can find a tutorial online about how to make a FM receiver with your specific SDR.
    • Hardware Tutorial
